WhereCOVID-19 Data Report

  • 1. Illinois Data

    COVID-19 data from Illinois Department of Public Health;

    • Counties level data (JSON); cumulative data; includes county name, confirmed_cases, total_tested, and demographic breakdown by age
      source: http://www.dph.illinois.gov/sitefiles/CountyDemos.json?nocache=1
    • Zip code level data (JSON); cumulative data; (includes zip code, confirmed_cases, total_tested, and demographic breakdown by age)
      source: http://www.dph.illinois.gov/sitefiles/COVIDZip.json?nocache=1
    • COVID historical test results (This dataset includes)
      • Cumulative characteristics of County; (includes County; confirmed_cases; total_tested; deaths; lat; long)
      • State level testing results by date; (includes testDate; total_tested; confirmed_cases; deaths)
      • State level demographics by age; (includes age_group, count, tested, deaths, demographics_by_race; )
      • State level demographics by race (includes race_description, count, tested, deaths)
      • State recovery data (updated weekly) (includes report_date, recovered_cases, recovered_and_deceased_cases, recovery_rates)
      • Historical characteristics for the county; updated daily; includes data going back from today to test date 03/17/2020 (includes County; confirmed_cases; total_tested; deaths; lat; long)
      source: http://www.dph.illinois.gov/sitefiles/COVIDHistoricalTestResults.json?nocache=1


    Others datasets:

    • Social Vulnerability index from CDC
      source: https://svi.cdc.gov/data-and-tools-download.html
    • HIV Carrier Density: AIDSVu
      source: https://aidsvu.org/
    • Accessibility measure from Kang et al (2020)
      source: https://medrxiv.org/cgi/content/short/2020.05.06.20093534v1

  • 2. United States

    COVID-19 data for US States and US County provided by New York Times; this data is updated daily;

    • US state level data (includes date,state,fips,cases,deaths)
      source: https://raw.githubusercontent.com/nytimes/covid-19-data/master/us-states.csv
    • US county level data (includes date,county,state,fips,cases,deaths)
      source: https://raw.githubusercontent.com/nytimes/covid-19-data/master/us-counties.csv

  • 3. World

    COVID-19 data provided by the World Health Organization

    • World level data updated daily (includes day, country, region, metrics (deaths, cumulative deaths, confirmed, cumulative confirmed)) includes historical data
      source: https://dashboards-dev.sprinklr.com/data/9043/global-covid19-who-gis.json
